アプリケーションの起動

この動画では様々なアプリケーションの起動方法について解説します。

1 プログラムメニューからのアプリケーションの起動

多くの場合、コンピュータにインストールされたアプリケーションはスタートメニューのプログラムメニューに登録されます。起動したいアプリケーションを探す場合はまずはこのプログラムメニューから探します。この例では「Microsoft Edge」というウェブブラウザを起動しています。 16秒 再生

2 検索ボックスを使ったアプリケーションの起動

「検索ボックス」でアプリケーションの名称を検索して起動することができます。この例では「Microsoft Edge」というウェブブラウザを検索して起動しています。 14秒 再生

3 スタートメニューとタスクバーへのアプリケーションのピン留め

普段良く起動するアプリケーションはスタートメニューやタスクバーにピン留めしておくと便利です。アプリケーションのアイコンを右クリックすることで「スタートにピン留めする」や「タスクバーにピン留めする」といったコンテキストメニューが表示されるのでここからそれぞれにピン留めすることができます。 20秒 再生

4 タスクバーを使ったアプリケーションの起動

普段良く起動するアプリケーションをタスクバーにピン留めしておくと簡単にアプリケーションを起動することができます。 7秒 再生

5 スタートメニューからのアプリケーションの起動

タスクバーは横幅に限りがあるのであまり沢山のアプリケーションをピン留めすることはできません。スタートメニューにピン留めしておくことでプログラムメニューよりは近い場所からアプリケーションを起動することができます。 7秒 再生
